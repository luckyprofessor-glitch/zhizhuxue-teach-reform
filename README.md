# 智助学｜AI社工游戏化课件平台

一个用于教改项目申报与原型演示的前端网页。

## 功能

- 上传常见课件：PDF、PPTX、DOCX、TXT、MD、HTML
- 自动抽取文本并重组为游戏化关卡
- 内置社工理论透镜：优势视角、赋能理论、生态系统理论、人在情境中、社会支持网络
- 内置 AI 风格功能：知识拆分、关键词提取、测验生成、即时反馈、学习建议
- 支持随机对照实验：自动分组、前测、后测、过程日志、JSON/CSV 导出
- 本地运行，无需后端即可演示

## 启动

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 说明

- 当前版本是浏览器端原型，AI 为本地轻量生成逻辑，便于教改展示与方法论论证。
- 如需正式上线，可继续接入 OCR、数据库、用户系统和真实大模型 API。
- 随机对照实验日志保存在浏览器本地存储中，导出后可用于后续统计分析。
