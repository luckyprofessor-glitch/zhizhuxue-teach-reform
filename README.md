# 智助学｜AI 游戏化课件自学工具

一个面向教改项目展示与原型验证的前端网页。

## 当前版本能做什么

- 上传常见课件：PDF、PPTX、DOCX、TXT、MD、HTML
- 自动抽取文本并重组为游戏化关卡
- 学生可直接进入自学界面，不需要分组或测量流程
- 内置智能体陪练：解释、举例、提示、出题、总结、下一步建议
- 新增教学型陪练模式：启发式、讲解式、挑战式、复盘式
- 新增分层提示：轻提示、中提示、关键提示
- 新增学习诊断、能力条、教学代理动作按钮
- 对 PPT / PDF 新增页级映射，生成结果会保留“第几页—第几页”对应关系
- 新增学习护照、班级／目标入口与最近学习档案，更接近学生产品形态
- 支持接入真实大模型：OpenAI 兼容接口、DeepSeek、OpenRouter、通义兼容模式、Kimi、SiliconFlow、自定义中转
- 支持用真实大模型优化关卡生成，也可只用来做学生对话陪练
- 融合社工理论透镜：优势视角、赋能理论、生态系统理论、人在情境中、社会支持网络
- 支持经验值、等级、徽章、主线关卡、终局任务
- 本地运行，无需后端即可演示；公开静态部署可直接打开链接体验

## 启动

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 真实 AI 使用说明

1. 打开网页后，在左侧“真实大模型设置”中开启真实大模型。
2. 选择一个提供方，或填写自定义 OpenAI 兼容接口。
3. 输入 Base URL、模型名、API Key。
4. 勾选：
   - 用真实 AI 优化关卡生成
   - 用真实 AI 接管学生对话陪练
5. 再上传课件并生成课程。

## 说明

- 当前版本是静态前端原型，API Key 只保存在当前浏览器本地存储。
- 如果某个提供方不允许浏览器直连，可能会遇到跨域限制，此时可把 Base URL 改成你自己的中转服务。
- 如需正式上线，可继续接入 OCR、数据库、账户系统和真实后端服务。
