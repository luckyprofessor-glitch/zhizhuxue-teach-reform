# 智助学｜AI 游戏化课件自学工具

一个面向教改项目展示与原型验证的前端网页。

## 当前版本做什么

- 上传常见课件：PDF、PPTX、DOCX、TXT、MD、HTML
- 自动抽取文本并重组为游戏化关卡
- 学生可直接进入自学界面，不需要分组或测量流程
- 内置智能体陪练：解释、举例、提示、出题、总结、下一步建议
- 融合社工理论透镜：优势视角、赋能理论、生态系统理论、人在情境中、社会支持网络
- 支持经验值、等级、徽章、主线关卡、终局任务
- 本地运行，无需后端即可演示

## 启动

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 说明

- 当前版本是浏览器端原型，AI 为本地轻量智能体逻辑，便于教改展示与交互演示。
- 如需正式上线，可继续接入 OCR、数据库、账户系统和真实大模型 API。
- 现在的重点是“学生自主学习工具”，不是实验分组平台。
